## 🎯 コンセプト

> 「あなたの競馬脳を育てるAI」
> 
> 
> UmaBrainは、あなたの競馬スタイルを理解し、最適な買い方を提案する**パーソナライズAI競馬コーチ**です。
> 
> 的中率を重視する人、回収率を狙う人、楽しみたい人──。
> 
> AIがあなたの過去の購入履歴やレース傾向を学習し、
> 
> “あなたらしい勝ち方”を共に作り上げます。
> 

---

## 💡 サービス概要

### 🏇 AIがあなたのスタイルを理解する

- 購入履歴、的中率、回収率、得意条件などを自動分析
- 「あなたはリスクを取る傾向」など、買い方の癖を見える化
- 個人ごとに最適化された“競馬脳”をAIが構築

### 💬 チャットでアドバイス

- AIチャットボットが「渡部さんならこの買い方」など具体的に助言
- 「的中率重視」「回収率重視」など、あなたの意図に応じた買い目提案
- レース直前・結果後のコメントも自動生成

### 📊 競馬データ × 機械学習

- JRA公式データを分析し、LightGBM / XGBoost で勝率・期待値を予測
- AIが学習を重ねることで、精度・パーソナライズが進化

### 💰 あなた専用の振り返りダッシュボード

- 自分の回収率・的中率をグラフで可視化
- 得意なコース・距離・騎手などを自動抽出
- 「AIが見たあなたの傾向レポート」を毎週自動生成

## ⚙️ 技術構成

### サーバーサイド
- Python / FastAPI  
- PostgreSQL / SQLAlchemy / Alembic  
- APScheduler（定期バッチ）  
- LightGBM / XGBoost（勝率予測モデル）

### フロントエンド
- Next.js / TypeScript / Tailwind CSS  
- Chart.js / Recharts  
- axios / SWR  

### インフラ・クラウド
- Docker / Docker Compose  
- Render（初期環境） → AWS ECS / RDS / S3（将来的移行）  
- GitHub Actions（CI/CD）
